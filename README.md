# java-scalatest
Demonstrates how to test Java code with [ScalaTest](http://scalatest.org/) and [ScalaCheck](http://scalacheck.org/).

## Java Classes to Test
* [User](src/main/java/org/sandbox/user/User.java)
* [UserStats](src/main/java/org/sandbox/user/UserStats.java)
* [UserDAO](src/main/java/org/sandbox/user/UserDAO.java)

## ScalaTest Classes
* [UserSpec](src/test/scala/org/sandbox/user/UserSpec.scala)
* [UserStatsSpec](src/test/scala/org/sandbox/user/UserStatsSpec.scala)
* [UserDAOSpec](src/test/scala/org/sandbox/user/UserDAOSpec.scala)
